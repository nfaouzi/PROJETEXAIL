# Unreal Engine Technical Test

## Description

This project was developed with **Unreal Engine 5.7.4** using **Blueprints** and the **Third Person Template**.

The objective of this project is to demonstrate basic Unreal Engine skills by creating an interactive environment where the player can move using keyboard inputs. The default template was used as a foundation and slightly customized by adding an extra platform to the level.

## Controls

This project uses the default **QWERTY** keyboard controls from the Unreal Engine Third Person Template.

**For QWERTY keyboards:**
- **W**: Move forward
- **A**: Move left
- **S**: Move backward
- **D**: Move right
- **Space**: Jump
- **Mouse**: Look around and rotate the camera

**For AZERTY keyboards:**
- **Z**: Move forward
- **Q**: Move left
- **S**: Move backward
- **D**: Move right
- **Space**: Jump
- **Mouse**: Look around and rotate the camera

## Technical Choices

- Unreal Engine 5.7.4
- Blueprint Visual Scripting
- Third Person Template
- Simple level with an additional platform
- Default movement and camera system

## How to Run

1. Clone or download this repository.
2. Open `PROJETEXAIL.uproject` with Unreal Engine 5.7.4.
3. Press **Play** to start the game.

## Project Structure

- `Content/` – Contains the game assets, Blueprints and maps.
- `Config/` – Unreal Engine configuration files.
- `PROJETEXAIL.uproject` – Unreal Engine project file.
